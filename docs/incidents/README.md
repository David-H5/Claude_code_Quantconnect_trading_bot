## Creating a New Incident Report

1. Copy the template: `cp ../processes/rca-template.md INC-XXX.md`
2. Assign next incident ID (INC-001, INC-002, etc.)
3. Fill out all required sections
4. Update this README with new incident
5. Create regression test in `tests/regression/`
