# Research Notebooks

This directory contains Jupyter notebooks for strategy research and analysis.

## Purpose

Use these notebooks to:
- Explore market data and patterns
- Test indicator combinations
- Analyze backtest results
- Visualize trading signals
- Prototype new strategies

## Getting Started

1. Install Jupyter: `pip install jupyter`
2. Start Jupyter: `jupyter notebook`
3. Create a new notebook or open existing ones

## Organization

- `exploration/`: Data exploration and analysis
- `backtests/`: Backtest result analysis
- `indicators/`: Indicator testing and validation
- `strategies/`: Strategy prototyping

## Best Practices

- Document your findings in markdown cells
- Include dates in notebook names
- Export successful strategies to `algorithms/` directory
- Version control notebooks (but consider clearing outputs before committing)
